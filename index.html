Machine Learning Project for Predicting Sales

(Project made for my Graduate Diploma Thesis at my University)

In this Project we Predicted the sales of a Supermarket chain in the USA called Bigmart. The prediction was for 30 days, using a dataset which had data for 4 years. We used XGBOOST model and Stacked Model (combining booth the OLS and SARIMAX models). In the end the Stacked Model was the best and that model is shown in the Picture.
